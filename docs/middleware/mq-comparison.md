```java
    /*
     * | 特性     | RocketMQ  | Kafka   | RabbitMQ  |
```

### 选型建议

```java
/**
 * 选型建议
 */
```

## 📖 面试真题

### Q1: 各 MQ 的特点？

**答：** RocketMQ 功能完善、Kafka 吞吐高、RabbitMQ 灵活。
