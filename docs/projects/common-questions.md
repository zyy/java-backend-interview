# 面试常见问题

> 项目相关问题

## 📖 常见问题

### 项目介绍

```
1. 介绍一下你最有成就感的项目
2. 项目中遇到的最大挑战是什么？
3. 你在项目中的角色？
4. 项目的技术选型依据？
```

### 技术深度

```
5. 为什么选择这个技术方案？
6. 有没有考虑其他方案？为什么放弃？
7. 系统的性能瓶颈在哪里？
8. 如何做性能优化的？
```

### 团队协作

```
9. 团队规模？你的角色？
10. 如何与产品、测试协作？
11. 有没有遇到技术分歧？如何解决？
```

---

**⭐ 重点：准备 3-5 个项目的详细介绍**
