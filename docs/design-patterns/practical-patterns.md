# 设计模式实战应用

> 设计模式在实际项目中的应用

## 🎯 面试重点

- Spring 中的设计模式
- JDK 中的设计模式

## 📖 Spring 中的设计模式

```java
/**
 * Spring 设计模式
 */
public class SpringPatterns {
    // 工厂模式
    /*
     * BeanFactory、FactoryBean
     */
    
    // 单例模式
    /*
     * Spring Bean 默认单例
     */
    
    // 代理模式
    /*
     * AOP
     */
    
    // 模板方法
    /*
     * JdbcTemplate、RestTemplate
     */
    
    // 策略模式
    /*
     * Resource 加载策略
     */
    
    // 责任链模式
    /*
     * Interceptor
     */
}
```

## 📖 JDK 中的设计模式

```java
/**
 * JDK 设计模式
 */
public class JdkPatterns {
    // 工厂模式
    /*
     * Calendar.getInstance()
     * NumberFormat.getInstance()
     */
    
    // 单例模式
    /*
     * Runtime.getRuntime()
     */
    
    // 装饰器模式
    /*
     * InputStream 系列
     */
    
    // 适配器模式
    /*
     * Arrays.asList()
     */
    
    // 观察者模式
    /*
     * EventListener
     */
}
```

## 📖 面试真题

### Q1: Spring 用到了哪些设计模式？

**答：** 工厂、单例、代理、模板方法、策略、责任链。
